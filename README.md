## 核心功能：不是花架子

**① 离线短信 AI 过滤**

用本地 AI 模型识别垃圾短信，全程不上传任何短信内容到服务器，连开发者自己都看不到你的短信。对隐私比较敏感的人，这点很重要。

**② 骚扰电话拦截**

内置上亿条号码库，每天自动更新。骚扰电话来了，系统自动比对，确认是骚扰号码直接静默拒接，还可以设置返回"空号"或"停机"提示音，让骚扰方觉得打错了，降低二次骚扰概率。

**③ 自定义规则**

可以按**地区**、**号段**、**关键词**设置拦截规则。比如你不想接任何上海来的陌生固话，直接设置地区拦截；某个号段反复来骚扰，加进黑名单一劳永逸。

苹果系统权限限制导致 iPhone 拦截效果始终不如安卓。拦截猫的解法是：**用安卓来帮 iPhone 拦截电话**。iPhone 来电时同步到安卓设备判断，让 Android 决定要不要拦截。双机党看到这里应该懂的。
